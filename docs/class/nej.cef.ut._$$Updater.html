• 构造参数

可选配置参数

名称类型描述
urlString更新服务器版本检查地址
filesObject需更新的文件ID和名称映射关系，如 {'core':'orpheus_install.exe'}
onstatechange onbeforeupdate onprogress

onstatechange

更新状态变化触发事件，状态值信息表：
状态值描述
0更新失败
1更新中
2暂停
3取消
4更新成功
5没有更新

• 输入

类型：Object
描述：状态信息
名称类型描述
stateNumber状态值，见描述
versionString版本信息
contentString版本更新内容

onbeforeupdate

开始更新之前触发事件

• 输入

类型：Object
描述：更新信息
名称类型描述
versionString版本信息
contentString更新内容

onprogress

更新过程触发事件

• 输入

类型：Object
描述：过程信息
名称类型描述
fileString当前下载的文件名
percentNumber当前文件下载进度，1-100
loadedNumber总进度，1-100
totalNumber总共需要下载文件个数
