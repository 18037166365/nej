• 输入

类型：String|Node
描述：执行节点
类型：Object
描述：构建卡片配置参数
名称类型描述
eventString触发显示卡片事件名称，默认为click事件
fixedBoolean是否固定位置，如果已固定位置则卡片显示位置不会自动计算
deltaObject四周偏移，默认全为0，{top:0,right:0,bottom:0,left:0}
alignString卡片位置，默认为'bottom left'
fitableBoolean是否需要调整卡片位置使其适应页面
noclearBoolean显示卡片之前是否不做卡片清理
toggledBoolean是否采用切换形式打开卡片，如果卡片已经显示则
onbeforeclickFunction点击之前触发事件，用以调整参数，输入卡片构造配置对象，{noclear:true,toggled:true}
