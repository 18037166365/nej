// 卡片显示的时候，根据target节点来定位,卡片位置固定
// align代表卡片的对齐方式
// delta代表偏移量
_card._$showByReference({
    align:'top right',
    target:document.body,
    delta:{top:10,right:20}
});
// 另外一种情况，根据鼠标点击来定位卡片位置
// 需要传入鼠标事件的event信息
// 鼠标传入的点的坐标为基准
// align表示，此点代表卡片的哪个位置
// delta代表偏移量
// 如果卡片超出可视区域范围，会参照windows桌面右键卡片布局
_card._$showByReference({
    event:_event,
    align:'top right',
    target:document.body,
    delta:{top:10,right:20}
});

类型：Object
描述：可选配置参数，已处理参数列表如下

alignObject卡片位置，默认为'top left'，{top:0,right:0,bottom:0,left:0} [ntb] 整体位置 | top/right/bottom/left 对齐方式 | top/bottom | left/right [/ntb]
fitableBoolean是否需要调整卡片位置使其适应页面
