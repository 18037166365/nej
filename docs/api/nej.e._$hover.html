nej.e._$hover(<String|Node>,<String>)

文件标识：{lib}base/element.js
依赖列表：{lib}base/constant.js,{lib}base/event.js,{lib}base/util.js,{patch}api.js,{patch}json.js
平台支持：          

• 简介

节点hover行为，高版本浏览器用:hover样式处理
样式举例
1
2
.page .element:hover,
.page .element.js-hover{background:#f00;}
页面结构举例
1
2
<!-- 使用data-hover指定hover效果的样式名称 -->
<div id="abc" data-hover="js-hover">aaaaa</div>
脚本举例
1
2
3
4
// 如果hover效果的样式名已经通过data-hover指定
nej.e._$hover('abc');
// 如果hover效果的样式名没有通过data-hover指定
nej.e._$hover('abc','js-hover');

• 输入

类型：String|Node
描述：节点
类型：String
描述：样式，默认为js-hover
