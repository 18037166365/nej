nej.e._$css3d(<String|Node>,<String>,<Object>)

文件标识：{lib}base/element.js
依赖列表：{lib}base/constant.js,{lib}base/event.js,{lib}base/util.js,{patch}api.js,{patch}json.js
平台支持：          

• 简介

设置3D变换，对于不支持3D的系统自动切换为2D变换
结构举例
1
<div id="css3d"></div>
脚本举例
1
2
3
4
var _e = NEJ.P("nej.e");
var _css3d = _e._$get('css3d');
// 进行css3d变换，对应css样式为-webkit-transform:rotate3d( 2, 1, 1, -75deg);
e._$css3d(_css3d,'rotate',{x:2,y:1,z:1,a:'-75deg'});

• 输入

类型：String|Node
描述：节点ID或者对象
类型：String
描述：变换类型，matrix/translate/scale/rotate
类型：Object
描述：变换值，{x:1,y:2,z:3,a:'30deg'}

类型：nej.e
