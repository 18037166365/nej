nej.e._$offset(<String|Node>,<String|Node>)

• 简介

计算两个节点之间的偏移量
页面结构举例
1
2
3
<div id="a" style="position:relative;padding:5px 0 0 10px;">
  <span id="b">123</span>
</div>
脚本举例
1
2
3
4
var _e = NEJ.P("nej.e");
// 计算节点b到节点a(外层需要定位属性)的距离，如果没有指定节点，默认计算的根节点
// _result : {x:10,y:5}
var _result = _e._$offset('b','a');

• 输入

类型：String|Node
描述：起始节点
类型：String|Node
描述：结束节点，没有该参数则计算到根节点

• 输出

类型：Object
描述：偏移量，如{x:234,y:987}
