• 简介

触发每个节点的对应事件, 同nej.v._$dispatchEvent，区别是参数类型自由度高一点 如果_event参数为String, 则视为简单事件触发如，click, 如果_event是个Ojbect,则会视为多重触发, 如{'click': param1, 'mouseover': param2} 如果_event是个Array, 则会对多个_event进行触发(公用一个options), 如['click','mouseover']Example:

//触发一个事件
_$trigger("click", params)
//一次触发多个事件(如果有参数，他们共用这个参数)
_$trigger(["click", "mouseover"], params)
// 触发多个事件有不同参数
_$trigger({
    "click": params1,
    "dblclick": params2
})

• 输入

类型：String|Array|Object
描述：_event 可以传入多种参数类型
类型：Whatever
描述：_options 同_$dispatchEvent的_options
