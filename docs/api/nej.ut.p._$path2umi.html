• 简介

路径转UMI
转换逻辑
    路径按照"?"或者"#"分割 识别私有模块，条件：第一个元素为"/"同时第二个元素以"?"开始 非私有模块返回第一个元素，私有模块返回用"?"合并的第一和第二个元素
转换举例
    /a/b              ->   /a/b    /a/b?a=aa         ->   /a/b    /a/b#a=aa         ->   /a/b    /a/b?a=aa#b=bb    ->   /a/b    /a/b#a=aa?b=bb    ->   /a/b    /a/b?/a/b         ->   /a/b    /a/b#/a/b         ->   /a/b    /?/a/b            ->   /?/a/b    /?/a/b?a=aa       ->   /?/a/b    /?/a/b#a=aa       ->   /?/a/b    /?/a/b?a=aa#b=bb  ->   /?/a/b    /?/a/b#a=aa?b=bb  ->   /?/a/b    /?a=aa            ->   /    /#a=aa            ->   /

• 输入

类型：String
描述：路径，可带查询参数

• 输出

类型：String
描述：模块UMI
