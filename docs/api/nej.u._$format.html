nej.u._$format(<Number|String|Date>,<String>)

文件标识：{lib}base/util.js
依赖列表：{lib}base/global.js,{lib}base/element.js
平台支持：          

• 简介

格式化时间，yyyy|yy|MM|cM|eM|M|dd|d|HH|H|mm|ms|ss|m|s|w
标识名称
yyyy四位年份，如2001
yy两位年费，如01
MM两位月份，如08
M一位月份，如8
dd两位日期，如09
d一位日期，如9
HH两位小时，如07
H一位小时，如7
mm两位分钟，如03
m一位分钟，如3
ss两位秒数，如09
s一位秒数，如9
ms毫秒数，如234
w中文星期几，如一
ct12小时制中文后缀，上午/下午
et12小时制英文后缀，A.M./P.M.
cM中文月份，如三
eM英文月份，如Mar
脚本举例
1
2
3
var _u = NEJ.P(nej.u);
// 根据格式输出时间，比如:2012-01-11,连接符可自定义
_u._$format(new Date(),'yyyy-MM-dd');

• 输入

类型：Number|String|Date
描述：时间
类型：String
描述：格式

• 输出

类型：String
描述：指定格式的时间串
