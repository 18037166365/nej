nej.$()._$add(<Node|Array|_$$NodeList>)

文件标识：E:/code/nej/src/util/chain/NodeList.js
依赖列表：
平台支持：          

• 简介

向内部节点集填入元素, 会处理好重复以及过滤的逻辑。这个也是$接口依赖的方法。
1
2
3
4
var $body = $("body")
$body._$add($("tbody")) //==> 添加tbody
$body._$add($("tbody")) //==> 什么都不会发生 因为重复了
$body._$add(document.body.childNodes) //==> 添加所有的body下的子节点,过滤掉不符合的

• 输入

类型：Node|Array|_$$NodeList
描述：_node 要添加的节点或节点集

• 输出

类型：_$$NodeList
描述：返回this
