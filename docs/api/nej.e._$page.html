// 构建一个pager控件
nej.e._$page('abc',{
    index:2,
    total:10,
    onchange:function(_event){}
});

• 输入

类型：String|Node
描述：分页器容器节点
类型：Object
描述：可选配置参数，参见nej.ut._$$Page控件的可选配置参数
类型：Object
描述：可选配置参数，已处理参数列表如下
名称类型描述
listArray页码节点列表【长度保持奇数】
eventString触发页码切换事件，默认为click
pbtnString|Node上一页按钮
nbtnString|Node下一页按钮
sbtnString|Node首页按钮
ebtnString|Node尾页按钮
indexNumber当前页码
totalNumber总页码数
selectedString选中样式，默认为js-selected
disabledString禁用样式，默认为js-disabled
clazzString需要做Page控件封装的节点标识，如没有此参数则选取父节点下的所有子节点

• 输出

类型：Function
描述：控件实例
