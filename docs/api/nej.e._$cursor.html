nej.e._$cursor(<String|Node>,<Number|Object>)

文件标识：{lib}base/element.js
依赖列表：{lib}base/constant.js,{lib}base/event.js,{lib}base/util.js,{patch}api.js,{patch}json.js
平台支持：          

• 简介

设置/获取光标位置在TEXTAREA中的位置脚本举例
1
2
3
4
5
6
7
8
// 设置光标选中内容
nej.e._$cursor('xxx',{start:5,end:10});
// 设置光标位置
nej.e._$cursor('xxx',8);
// 获取光标位置
var _position = nej.e._$cursor('xxx');
// _position.start 光标起始位置
// _position.end   光标结束位置

• 输入

类型：String|Node
描述：TEXTAREA节点
类型：Number|Object
描述：待设置光标的位置，如果起始位置和结束位置一致则输入数值即可
名称类型描述
startNumber起始位置
endNumber结束位置，没有end则表示与start相同

• 输出

类型：Object
描述：光标位置，{start:0,end:10}
