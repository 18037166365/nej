nej.e._$tab(<String|Node>,<Object>)

文件标识：{lib}util/tab/tab.js
依赖列表：{lib}base/event.js,{lib}base/element.js,{lib}util/event.js
平台支持：          

• 简介

将某个父容器下的相关节点构造成一个TAB控件
1
2
3
4
5
6
// 构建一个tab控件
nej.e._$tab('abc',{onchange:function(){}});
// 调用tab控件接口
nej.e._$tab('abc')._$go(5);
// 回收tab控件
nej.e._$tab('abc')._$recycle();

• 输入

类型：String|Node
描述：TAB控件父容器
类型：Object
描述：可选配置参数，参见nej.ut._$$Tab控件的可选配置参数
名称类型描述
clazzString需要做TAB控件的节点标识，如没有此参数则选取父节点下的所有子节点

• 输出

类型：nej.ut._$$Tab
