nej.$()._$style(<String|Object|Array>,<String>)

文件标识：E:/code/nej/src/util/chain/NodeList.js
依赖列表：
平台支持：          

• 简介

获取节点样式或者设置节点样式, 这个接口的表现与jQuery的css方法一致, 根据参数不同有不同的表现比如:
1
2
3
4
$('li')._$style(name) //相当于_$getStyle 返回样式值
$('li')._$style([name1,name2...]) //相当于多重_$getStyle 返回一个Object(如{"height:20px, width:30px..."})
$('li')._$style(name, value) // 相当于setStyle 返回this
$('li')._$style(obj) //相当于多重版setStyle(即原_$style) 返回this

• 输入

类型：String|Object|Array
描述：_key 可以是String(单取值或设置)，一个对象(多重赋值)，一个数组(多重取值)
类型：String
描述：_value 样式值

• 输出

类型：_$$Nodelist|String|Object
描述：setter操作返回_$$NodeList，单重取值返回String，多重取值返回样式属性为键的Object，表现与jQuery的css接口一致
