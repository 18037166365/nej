NEJ.EX(<Object>,<Object>)

文件标识：{lib}base/global.js
依赖列表：
平台支持：          

• 简介

根据原始对象属性，从目标对象拷贝相应值
脚本举例
1
2
3
4
5
var _obj0 = {a:0,b:1},
    _obj1 = {a:"a",b:"b",c:"c"};
// 根据_obj0的属性,从_obj1拷贝属性到_obj0中,如果属性的value是null不会被拷贝
// 结果是_obj0.a = "a",_obj.b = "b",没有拷贝c属性;
var _obj = NEJ.EX(_obj0,_obj1);
