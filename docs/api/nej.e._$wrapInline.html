nej.e._$wrapInline(<String|Node>,<Object>)

文件标识：{lib}base/element.js
依赖列表：{lib}base/constant.js,{lib}base/event.js,{lib}base/util.js,{patch}api.js,{patch}json.js
平台支持：          

• 简介

内联元素增加定位封装
加入以下html代码
1
<input type="text" id="abc"/>
在执行了以下代码后
1
var node = nej.e._$wrapInline('abc');
最后生成以下结构
1
2
3
4
5
<span style="position:relative;zoom:1">
  <input type="text" id="abc"/>
  <!-- 此api返回以下这个节点 -->
  <span style="position:absolute;top:0;left:0;"></span>
</span>
应用举例
1
2
3
4
5
6
7
8
9
var input = nej.e._$get('abc');
// 返回容器的样式名称
// 通过这个样式名称可以取到一个绝对定位的样式名 class+'-show'
var node = nej.e._$wrapInline(input,{
    tag:'label',
    clazz:'js-xxx'
});
// 可以在返回的节点里添加想要显示的结构
node.innerHTML = '<span>aaa</span><span>aaa</span>';

• 输入

类型：String|Node
描述：内联节点
类型：Object
描述：据对定位节点配置信息
名称类型描述
tagString标记名称，默认span
clazzString样式名称
nidString节点识别样式名，这个会被添加到样式中作为标识

• 输出

类型：Node
描述：绝对定位的节点
